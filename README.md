# Transaction Cleaner (Electron Hello World)

This is a minimal Electron desktop app you can run locally and package into a distributable.

## Prereqs

- Node.js + npm installed

## Run (dev)

```bash
npm install
npm start
```

## Package (build)

- Create unpacked app folder:

```bash
npm run pack
```

- Create an installable / distributable:

```bash
npm run dist
```

Output goes to `dist/`.
